## SUS Score Calculation Method

SUS uses 5 questions (scale 1-5):
- Odd questions (Q1, Q3, Q5) — positive: score - 1
- Even questions (Q2, Q4) — negative: 5 - score
- Sum all results × 5 = final score out of 100

**Questions Asked:**
- Q1: I think that I would like to use this system frequently
- Q2: I found the system unnecessarily complex
- Q3: I thought the system was easy to use
- Q4: I think that I would need the support of a technical person to use this system
- Q5: I found the various functions in this system were well integrated
